# Relation: Define

## Supported Pattern
```yaml
name: Define
```
### Examples
- Module Level Definition
```python
// test_module_level_define.py

class Base:
    ...

class Inherit(Base):
    ...

def func1():
    return 0

x = 1

y: int = 1

t1, t2 = 1, 2

(t3 := 1)


```

